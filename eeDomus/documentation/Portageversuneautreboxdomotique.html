Portage vers une autre box domotique


Rien n'interdit le portage de ce plug-in vers une autre box domotique.

Les seules choses à porter (remplacer) sont:

    • Les 4 fonctions de dialogues avec la box domotique qui sont utilisées par le plug-in, toutes rassemblées dans le fichier <plug-in>/ node_modules/lib-avatar/eeDomus.js
    • L'authentification et les clés d'accès à la box domotique:
      • Sauvegardés dans le fichier eeDomus.prop, clé API
      • Le contrôle de ces clés dans les fichiers <plug-in>/eeDomus.js et <plug-in>/assets/html/eeDomus.js
    • L'image du node dans l'interface
    • Le nom de la fenêtre de gestion des widgets et quelques messages.


Les puristes pourront s'amuser à modifier toutes les références au plug-in eeDomus, vois renommer les fichiers mais ce n'est pas du tout indispensable.



Les fonctions dans le fichier <plug-in>/ node_modules/lib-avatar/eeDomus.js:


getPeriphCaract (periph_id || all)

Paramètre: l'ID d'un périphérique ou "all" pour tous les périphériques

Doit retourner: Un objet contenant les caractéristiques du périphérique ou un tableau d'objets de tous les périphériques, suivant le paramètre d'entré

Exemple de réponse

{
 "name": "Mouvement Atelier",
 "last_value": "100",
 "last_value_text": "Mouvement",
 "battery": "60",
 "last_value_change": "2010-03-29 14:51:22"
}



Description des champs de la réponse

Nom

Type

Description

name

string

Nom du périphérique

last_value

string

Dernière valeur du périphérique

last_value_text

string

Description de la valeur du périphérique si disponible

battery

string

Niveau de la batterie en pourcentage

last_value_change

string

Date/heure correspondant à cette dernière valeur (Au format AAAA-MM-JJ HH:MM:SS)




getPeriphValueList (periph_id)

Paramètre: l'ID d'un périphérique

Doit retourner: Un objet contenant les valeurs possibles d'un périphérique.

Exemple de réponse

{
  "periph_id": 72762,
  "values": [
          {
              "value": "0",
              "description": "Off"
          },
          {
              "value": "100",
              "description": "On"
          }
  ]
}




set (periph_id, value)

Paramètre: l'ID et la valeur à définir pour le périphérique

Doit retourner: rien ou l'erreur
