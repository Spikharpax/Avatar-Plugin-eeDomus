Exécuter une macro

Les périphériques eeDomus de type "listes de valeurs" peuvent exécuter des macros qui leurs sont propres, par exemple, une macro qui va éteindre progressivement la lumière pour un variateur.


Il n'existe pas de fonction de l'API eeDomus qui permet de récupérer les macros associées à un périphérique.

Néanmoins, en exécutant une petite manipulation (très simple), le plug-in est capable de les gérer.


Prenons comme exemple un périphérique de type "Lampe" avec variateur de lumière pour lequel j'ai une macro eeDomus qui éteint progressivement la lumière:




  1. Je crée une nouvelle valeur (onglet valeurs) que je nomme comme je veux (ex: Graduel)




  1. Je récupère le code API de la macro et la valeur brute et je l'ajoute en "Note utilisateur" du périphérique (Onglet périphérique) sous la forme valeur brute,Code API:



  1. Dans l'interface du plug-in, définissez les listes et sélectionnez "Peut déclencher une macro":




  1. Si vous créez une règle associée à une valeur qui exécute une macro, cochez aussi la case "Peut exécuter une macro".
  2. Lorsque vous cliquerez sur la valeur, la macro sera exécutée.
